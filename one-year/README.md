# ONE YEAR

**What does your year really buy?**

ONE YEAR is an interactive web app that reframes economic discussions using a unit of measurement: **1Y (One Year) = one median worker's annual salary.** Instead of showing prices in dollars, we show costs as fractions of a year's labor.

## What is 1Y?

1Y = one median worker's annual salary. We use it to compare costs across decades because dollar amounts are meaningless across time. A house cost 2.3 years of median income in 1950. It costs 4.9 years in 2025.

## Tech Stack

- React 19 with Vite
- Tailwind CSS
- Static JSON data

## Getting Started

```bash
git clone <repo>
cd one-year
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
npx vite preview
```
